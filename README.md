# dotfiles
The dotfiles I use for development.
How to install: https://harfangk.github.io/2016/09/18/manage-dotfiles-with-a-git-bare-repository.html

I use the following tools:
- Terminals
    - [iTerm2](https://iterm2.com/) (great default terminal for mac)
    - [alacritty](https://github.com/alacritty/alacritty) (for when I need extra speed)
  - with [Nerd Fonts](https://www.nerdfonts.com/) (for nice icons)
- [Neovim](https://neovim.io/) (my favorite editor)
  - with [packer.nvim plugins](https://github.com/wbthomason/packer.nvim)
- [zsh](https://www.zsh.org/) (my favorite shell)
  - with [oh-my-zsh](https://ohmyz.sh/)
- [tmux](https://github.com/tmux/tmux/wiki) (my favorite terminal multiplexer)
  - with [tpm](https://github.com/tmux-plugins/tpm)
- [Rust CLI tools](https://crates.io/)
  - [bat](https://github.com/sharkdp/bat) (better `cat`)
  - [delta](https://github.com/dandavison/delta) (better git highlighting)
  - [dust](https://github.com/bootandy/dust) (better `du`)
  - [exa](https://github.com/ogham/exa) (better `ls`)
  - [fd](https://github.com/sharkdp/fd) (better `find`)
  - [gitui](https://github.com/extrawurst/gitui) (git TUI)
  - [inlyne](https://github.com/trimental/inlyne) (browserless markdown preview)
  - [monolith](https://github.com/y2z/monolith) (save web pages into single html)
  - [ouch](https://github.com/ouch-org/ouch) (compress and decompress files)
  - [porsmo](https://crates.io/crates/porsmo) (pomodoro timer)
  - [procs](https://github.com/dalance/procs) (better `ps`)
  - [rip](https://github.com/nivekuil/rip) (better `rm`)
  - [rg](https://github.com/BurntSushi/ripgrep) (better `grep`)
  - [topgrade](https://github.com/topgrade-rs/topgrade) (helps upgrade CLI tools)
  - [sd](https://github.com/chmln/sd) (better `sed`)
  - [speedtest-rs](https://github.com/nelsonjchen/speedtest-rs) (better `ping` that uses speedtest.net)
  - [tldr](https://github.com/dbrgn/tealdeer) (better `man`)
  - [zoxide](https://github.com/ajeetdsouza/zoxide) (`cd` but from anywhere)
